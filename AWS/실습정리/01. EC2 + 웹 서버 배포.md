## 1️⃣ EC2 + 웹 서버 배포 (기본기 ⭐)

### 📌 내용
- EC2 인스턴스 생성
- Apache 또는 Nginx 설치
- HTML 파일 배포

### 🎯 배우는 것
- 서버 접속 (SSH)
- 리눅스 기본 명령어
- 웹 서비스 동작 구조

---
1️⃣ EC2 인스턴스 생성

2️⃣ EC2 접속 (SSH)
ex. ssh -i key.pem ec2-user@퍼블릭IP

3️⃣ 웹 서버 설치
📌 Apache 설치
```
sudo yum update -y
sudo yum install -y httpd
```

📌 서버 실행
```
sudo systemctl start httpd
sudo systemctl enable httpd
```
👉 재부팅해도 자동 실행

4️⃣ HTML 파일 배포
```
echo "<h1>Hello AWS!</h1>" | sudo tee /var/www/html/index.html
```


5️⃣ 브라우저에서 확인
```
http://퍼블릭IP
```
