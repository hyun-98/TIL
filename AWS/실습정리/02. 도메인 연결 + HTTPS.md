## 2️⃣ 도메인 연결 + HTTPS (강추 ⭐)

### 📌 내용
- 도메인 구매 (연 1~2만원)
- DNS 설정 (A 레코드)
- Let's Encrypt로 HTTPS 적용

### 🎯 배우는 것
- 도메인 → 서버 연결 구조
- HTTPS 인증서 개념

---
1️⃣ 도메인 구매

👉 아무 곳에서 사도 OK

가비아
후이즈
Namecheap
GoDaddy

👉 예:

myserver123.com
2️⃣ DNS 설정 (핵심 ⭐)

👉 도메인 → EC2 IP 연결

📌 설정할 것

DNS 관리에서:

타입	값
A 레코드	EC2 퍼블릭 IP
📌 예시
Type: A
Name: @
Value: 43.203.xxx.xxx

👉 저장 후 1~10분 기다림

3️⃣ 연결 확인

👉 브라우저에서:

http://내도메인

👉 잘 되면:
→ EC2 페이지 뜸

4️⃣ HTTPS 적용 (Let’s Encrypt)

👉 무료 인증서 사용

📌 EC2 접속 후 실행
1) certbot 설치
sudo yum install -y certbot python3-certbot-apache
2) 인증서 발급
sudo certbot --apache

👉 진행 중:

이메일 입력
도메인 입력
HTTPS 적용 선택
🎉 결과

👉 자동으로:

HTTPS 설정됨
Apache 설정 변경됨
5️⃣ HTTPS 확인
https://내도메인

👉 주소창:
🔒 표시 뜨면 성공

🔄 자동 갱신 (중요)

👉 Let’s Encrypt는 90일짜리

📌 테스트
sudo certbot renew --dry-run

👉 문제 없으면 자동 갱신됨
