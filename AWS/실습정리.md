# 🚀 AWS 프리티어 실습 로드맵

> 💡 목표: 비용을 최소화하면서 AWS 실무 감각 익히기

---

## ✅ 기본 원칙

- ❌ NAT Gateway 사용하지 않기 (비용 발생)
- ❌ 리소스 방치 금지
- ✅ Public Subnet 중심으로 실습
- ✅ 실습 후 반드시 리소스 삭제

---

## 1️⃣ EC2 + 웹 서버 배포 (기본기 ⭐)

### 📌 내용
- EC2 인스턴스 생성
- Apache 또는 Nginx 설치
- HTML 파일 배포

### 🎯 배우는 것
- 서버 접속 (SSH)
- 리눅스 기본 명령어
- 웹 서비스 동작 구조

---

## 2️⃣ 도메인 연결 + HTTPS (강추 ⭐)

### 📌 내용
- 도메인 구매 (연 1~2만원)
- DNS 설정 (A 레코드)
- Let's Encrypt로 HTTPS 적용

### 🎯 배우는 것
- 도메인 → 서버 연결 구조
- HTTPS 인증서 개념

---

## 3️⃣ S3 정적 웹 호스팅

### 📌 내용
- HTML/CSS 파일 업로드
- 정적 웹사이트 배포

### 👍 장점
- 서버 필요 없음
- 비용 거의 0
- 빠른 속도

---

## 4️⃣ CloudFront CDN 적용

### 📌 내용
- S3 앞단에 CDN 구성

### 🎯 배우는 것
- 캐싱(Cache)
- 글로벌 콘텐츠 전송 구조

---

## 5️⃣ IAM (권한 관리)

### 📌 내용
- 사용자(User) 생성
- 권한(Policy) 설정
- Role 개념 이해

### 🎯 중요성
- 보안 핵심
- 면접 단골 질문

---

## 6️⃣ 보안 그룹 실습

### 📌 내용
- 포트 열기/닫기
- SSH 접근 제한

### 🎯 배우는 것
- 네트워크 보안 기본
- 인바운드/아웃바운드 규칙 이해

---

## 7️⃣ 간단한 3-Tier 구조 (NAT 없이)

User → EC2 (Public)


### 📌 내용
- 기본적인 서비스 흐름 이해

### ⚠️ 참고
- 실제 3-tier 구조는 아님 (개념 학습용)

---

## 8️⃣ Auto Scaling + ALB (주의 ⚠️)

### 📌 내용
- EC2 여러 대 구성
- Application Load Balancer 연결

### 🎯 배우는 것
- 트래픽 분산
- 고가용성 구조

### ⚠️ 주의
- 실습 후 반드시 삭제 (비용 발생 가능)

---

## 9️⃣ RDS (DB) 실습

### 📌 내용
- MySQL 데이터베이스 생성
- EC2에서 DB 연결

### 🎯 배우는 것
- 서버 ↔ DB 연결 구조

### ⚠️ 주의
- 프리티어: 월 750시간 제한

---

## 🔟 SSM 접속 (SSH 없이)

### 📌 내용
- AWS Systems Manager 사용
- 키 없이 EC2 접속

### 🎯 배우는 것
- 최신 서버 접속 방식
- 보안 강화 방법

---

## 💰 비용 관리 체크리스트

- [ ] 실습 후 EC2 종료 (Stop 또는 Terminate)
- [ ] RDS 삭제
- [ ] Load Balancer 삭제
- [ ] 불필요한 리소스 정리
- [ ] NAT Gateway 생성 금지

---

## 🎯 추천 학습 순서

1. EC2 + 웹 서버 배포  
2. 도메인 + HTTPS  
3. S3 정적 호스팅  
4. CloudFront  
5. IAM / 보안 그룹  
6. ALB (짧게 실습)  
7. RDS
