
EC2 = 내가 원하는 사양의 서버를 클릭 몇 번으로 만드는 서비스

EC2의 구성요소
1. 인스턴스 (Instance)
👉 실제로 빌리는 “가상의 컴퓨터”
CPU, RAM 선택 가능
예: t2.micro, t3.large 등

2. AMI (이미지)
👉 서버의 기본 상태 (운영체제)
- Ubuntu
- Amazon Linux
- Windows Server
👉 “컴퓨터 처음 켰을 때 상태”라고 보면 됨

3. 스토리지 (EBS)
👉 하드디스크 역할
- 데이터 저장
- 서버 꺼도 유지됨

4. 보안 그룹 (Security Group)
👉 방화벽
- 어떤 IP가 접속 가능한지 설정
예: 22번 포트(SSH), 80번(웹)

5. 키 페어 (Key Pair)
👉 서버 로그인용 열쇠 🔑
- 비밀번호 대신 사용
- SSH 접속할 때 필요
