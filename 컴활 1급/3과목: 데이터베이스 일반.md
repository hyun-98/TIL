👉 데이터베이스 핵심 정리

---

# 📌 1. 데이터베이스 기본 개념

## 🧠 DB란?

* 데이터를 체계적으로 저장/관리하는 시스템
* 중복 최소화 + 효율적 검색

## ✔️ DBMS

👉 데이터베이스 관리 시스템

* 데이터 저장, 수정, 삭제 관리

---

# 📊 2. 관계형 데이터베이스 (RDB)

## ✔️ 특징

* 테이블(표) 구조
* 행(Row) + 열(Column)

## ✔️ 핵심 용어

* 테이블 = Relation
* 행 = Tuple
* 열 = Attribute

---

# 🔑 3. 키(Key) 개념 (🔥 중요)

| 종류                | 의미           |
| ----------------- | ------------ |
| 기본키 (Primary Key) | 유일 + NULL 불가 |
| 외래키 (Foreign Key) | 다른 테이블 연결    |
| 후보키               | 기본키 후보       |
| 슈퍼키               | 유일성 포함 집합    |

---

# 🧩 4. SQL 기본 (🔥 최중요)

## ✔️ 조회

```sql id="2p1n7q"
SELECT * FROM table;
```

## ✔️ 조건

```sql id="k8x1f9"
SELECT * FROM table WHERE age > 20;
```

## ✔️ 삽입

```sql id="d2x9qv"
INSERT INTO table VALUES (...);
```

## ✔️ 수정

```sql id="v9k3ab"
UPDATE table SET name='A';
```

## ✔️ 삭제

```sql id="q7m1ld"
DELETE FROM table;
```

---

# 🔗 5. JOIN (테이블 연결)

## ✔️ INNER JOIN

* 공통 데이터만

## ✔️ LEFT JOIN

* 왼쪽 기준 + 매칭 없으면 NULL

## ✔️ RIGHT JOIN

* 오른쪽 기준

---

# 📐 6. 정규화 (Normalization)

👉 데이터 중복 제거 과정

## ✔️ 단계

* 1NF: 원자값
* 2NF: 부분 종속 제거
* 3NF: 이행 종속 제거

---

# 📂 7. 트랜잭션 (Transaction)

👉 하나의 작업 단위

## ✔️ 특징 (ACID)

* Atomicity (원자성)
* Consistency (일관성)
* Isolation (독립성)
* Durability (지속성)

---

# 📊 8. 데이터 모델

* 개념적 모델 (ERD)
* 논리적 모델 (테이블)
* 물리적 모델 (저장 구조)

---

# 🔥 9. 자주 나오는 핵심

👉 시험 단골

* PRIMARY KEY vs FOREIGN KEY
* SELECT / WHERE
* JOIN 종류
* 정규화 단계
* ACID 개념

---

# 🧠 초압축 암기

👉 SELECT = 조회
👉 INSERT = 추가
👉 UPDATE = 수정
👉 DELETE = 삭제
👉 PK = 유일값
👉 FK = 연결
👉 JOIN = 테이블 합침
