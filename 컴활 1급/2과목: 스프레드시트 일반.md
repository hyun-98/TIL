
# 📊 컴활 1급 필기 – 2과목 스프레드시트 일반 (1페이지 요약)

👉 Microsoft Excel 중심 핵심 정리

---

# 📌 1. 엑셀 기본 개념

## 🧾 워크북 / 워크시트

* 워크북: 파일 전체 (엑셀 파일 1개)
* 워크시트: 시트 1장 (Sheet1, Sheet2)

## 📍 셀 구조

* 행(Row) + 열(Column)
* 셀 주소: A1, B3 등

---

# 📌 2. 참조 방식 (무조건 출제)

## ✔️ 상대참조

* A1 → 복사하면 자동 변경

## ✔️ 절대참조

* $A$1 → 고정

## ✔️ 혼합참조

* $A1 / A$1 → 행 또는 열 고정

---

# 📌 3. 핵심 함수 (시험 핵심🔥)

## 🧮 기본 함수

* SUM: 합계
* AVERAGE: 평균
* MAX / MIN: 최대 / 최소

---

## 🧠 논리 함수

* IF: 조건

```excel
=IF(A1>10,"합격","불합격")
```

---

## 🔎 조회 함수 (최중요)

### ✔️ VLOOKUP

* 세로 검색
* 왼쪽 → 오른쪽

```excel
=VLOOKUP(값, 범위, 열번호, FALSE)
```

---

### ✔️ HLOOKUP

* 가로 검색

---

## 📊 고급 함수

* INDEX + MATCH (VLOOKUP 대체)
* COUNT / COUNTA / COUNTIF

---

# 📌 4. 데이터 처리 기능

## 🔽 정렬

* 오름차순 / 내림차순

## 🔍 필터

# 📌 5. 차트 기능

👉 데이터 시각화

---

# 📌 6. 데이터 입력/서식

* 숫자 / 날짜 / 텍스트
* 사용자 지정 서식
* 조건부 서식 (중요🔥)

# 📌 7. 자주 나오는 포인트

# 📊 Excel 함수 암기표 (컴활 1급 시험용 30개)

👉 Microsoft Excel 핵심 함수 정리

---

# 🧮 1. 기본 계산 함수

| 함수      | 의미  | 핵심      |
| ------- | --- | ------- |
| SUM     | 합계  | 전체 더하기  |
| AVERAGE | 평균  | 산술 평균   |
| MAX     | 최대값 | 가장 큰 값  |
| MIN     | 최소값 | 가장 작은 값 |
| PRODUCT | 곱   | 전체 곱셈   |

---

# 🔢 2. 개수 관련 함수

| 함수         | 의미    | 핵심      |
| ---------- | ----- | ------- |
| COUNT      | 숫자 개수 | 숫자만     |
| COUNTA     | 전체 개수 | 빈칸 제외   |
| COUNTBLANK | 빈칸 개수 | 공백 셀    |
| COUNTIF    | 조건 개수 | 조건 1개   |
| COUNTIFS   | 다중 조건 | 조건 여러 개 |

---

# 🧠 3. 논리 함수

| 함수  | 의미    | 핵심           |
| --- | ----- | ------------ |
| IF  | 조건    | TRUE / FALSE |
| AND | 모두 만족 | 전부 TRUE      |
| OR  | 하나라도  | 하나 TRUE      |
| NOT | 반전    | TRUE → FALSE |

---

# 🔍 4. 검색/조회 함수 (🔥 핵심)

| 함수      | 의미     | 핵심     |
| ------- | ------ | ------ |
| VLOOKUP | 세로 검색  | 왼쪽→오른쪽 |
| HLOOKUP | 가로 검색  | 위→아래   |
| INDEX   | 위치값 반환 | 좌표 기반  |
| MATCH   | 위치 찾기  | 몇 번째인지 |

---

# 🧩 5. 텍스트 함수

| 함수     | 의미      | 핵심      |
| ------ | ------- | ------- |
| LEFT   | 왼쪽 자르기  | 앞부분     |
| RIGHT  | 오른쪽 자르기 | 끝부분     |
| MID    | 중간 자르기  | 시작+길이   |
| LEN    | 길이      | 글자 수    |
| CONCAT | 연결      | 문자열 합치기 |

---

# 📅 6. 날짜/시간 함수

| 함수    | 의미    | 핵심      |
| ----- | ----- | ------- |
| TODAY | 오늘 날짜 | 자동 업데이트 |
| NOW   | 현재 시간 | 날짜+시간   |
| YEAR  | 연도    | 연도 추출   |
| MONTH | 월     | 월 추출    |
| DAY   | 일     | 일 추출    |

---

# 📊 7. 통계 함수

| 함수    | 의미   | 핵심       |
| ----- | ---- | -------- |
| RANK  | 순위   | 등수       |
| LARGE | 큰 값  | n번째 큰 값  |
| SMALL | 작은 값 | n번째 작은 값 |
| STDEV | 표준편차 | 데이터 분산   |
| VAR   | 분산   | 데이터 퍼짐   |
