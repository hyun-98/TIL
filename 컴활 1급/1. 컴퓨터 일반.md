

👉 컴퓨터활용능력 1급 필기 1과목 핵심 정리

---

# 💻 1. 컴퓨터 시스템 기본

## 🧠 중앙처리장치 (CPU)

👉 CPU

* 컴퓨터 “두뇌”
* 구성:

  * ALU (연산장치)
  * CU (제어장치)
* 성능: 클럭 속도(GHz)

---

## 🧠 기억장치

### 🔹 주기억장치

* RAM (휘발성)
* ROM (비휘발성)

### 🔹 보조기억장치

* HDD / SSD
* 데이터 영구 저장

---

# 🖥 2. 운영체제 (OS)

👉 Windows 기준 출제

## ✔️ 역할

* 하드웨어 관리
* 프로그램 실행
* 사용자 인터페이스 제공

## ✔️ 기능

* 프로세스 관리
* 메모리 관리
* 파일 시스템 관리

---

# 🌐 3. 네트워크 기초

## ✔️ 기본 개념

* IP 주소: 컴퓨터 주소
* DNS: 도메인 → IP 변환
* MAC 주소: 물리적 주소

## ✔️ 프로토콜

* TCP: 신뢰성 (연결형)
* UDP: 속도 (비연결형)

---

# 🔐 4. 정보보안 기초

## ✔️ 기본 목표 (CIA)

* Confidentiality (기밀성)
* Integrity (무결성)
* Availability (가용성)

## ✔️ 공격 유형

* 바이러스
* 웜
* 트로이목마
* 피싱

---

# 📁 5. 데이터 단위

* bit (최소 단위)
* byte = 8bit
* KB → MB → GB → TB

👉 1024 단위로 증가

---

# ⚙️ 6. 입출력 장치

## ✔️ 입력장치

* 키보드
* 마우스
* 스캐너

## ✔️ 출력장치

* 모니터
* 프린터

---

# 🧠 7. 자주 나오는 핵심

🔥 시험 단골

* CPU 구성 (ALU / CU)
* RAM vs ROM
* TCP vs UDP
* IP vs DNS
* 바이러스 vs 웜 vs 트로이목마

---

# 🧠 초압축 암기

👉 CPU = 연산
👉 RAM = 임시 저장
👉 OS = 관리 시스템
👉 TCP = 안정
👉 UDP = 속도
👉 DNS = 이름 → IP

---

# 💡 한 줄 정리

👉 **컴퓨터 일반은 “CPU + 메모리 + 운영체제 + 네트워크 + 보안 기초”를 묻는 과목이다**
