## 🛡️ 내부 보안 솔루션

```
[인터넷]
    ↓
[Firewall]                 ← 허용되지 않은 포트/IP 차단
    ↓
[IPS]                      ← 포트 스캔, Malware, DDoS, 보안 취약점(이전 version)
    ↓
[WAF]                          ← 외부 보안
    ↓
[회사 네트워크]
    ↓
[NAC]                      ← 네트워크 입구  # 네트워크에 접속 O X
    ↓
    ├── 💻 직원 PC
    ├── 🖥️ 서버
    └── 📱 스마트폰
         ↓
    [EDR / 패치관리]        ← 기기 안  # 실시간 감시 (악성코드 탐지) / 최신 버전으로 업데이트
         ↓
    [ACL / RBAC]           ← 파일/시스템 앞  # 파일 접근 권한
         ↓
    [DLP]                  ← 데이터 나가는 길목  # 중요 파일 유출 금지
         ↓
[이상행동 탐지 / SIEM (전체 로그 통합 분석)]             ← 전체를 위에서 감시
```

```
7계층 ── WAF, DLP, DRM, ACL/RBAC
          "내용을 읽고 판단"

4~6계층 ── IPS (일부), DRM (암호화)
            "연결 방식과 데이터 형식 검사"

1~3계층 ── 방화벽, NAC
            "IP, 포트, MAC주소로 판단"
```
