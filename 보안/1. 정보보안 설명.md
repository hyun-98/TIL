# 🧠 1. 보안 “전략 / 관리” 영역 (큰 개념)

👉 **조직이 어떻게 대응하냐**

## 📌 위험 관리 전략 (핵심 4가지)

* **순응 (Accept)** → 그냥 사용
* **회피 (Avoid)** → 위험 자체 제거
* **전가 (Transfer)** → 외주/보험
* **감소 (Mitigate)** → 보안 강화

👉 💡 한 줄 정리
**“버릴까 / 넘길까 / 줄일까 / 그냥 둘까”**

---

## 📌 PIA (개인정보 영향평가)

* 시스템 도입 시 개인정보 리스크 분석

## 📌 ISMS-P

* 기업의 보안 관리체계 인증

👉 💡 묶음 포인트
👉 **“관리/제도/인증” 영역**

---

## 📌 GDPR

* EU 개인정보 규제

👉 💡 같이 묶기
→ **법 + 규제**

---

# 🔐 2. 개인정보 / 데이터 보호

👉 **데이터 자체 보호 개념**

## 📌 가명정보 vs 익명정보

| 구분   | 특징     |
| ---- | ------ |
| 가명정보 | 재식별 가능 |
| 익명정보 | 재식별 불가 |

👉 💡 핵심

* 가명 = 위험 있음
* 익명 = 안전

---

## 📌 위치정보 법

### 🔹 위치정보사업

* 수집/제공 (허가 필요)

### 🔹 위치기반서비스사업

* 활용 (신고)

👉 💡 핵심
👉 **수집 = 사업 / 활용 = 서비스**

---

# 🌐 3. 네트워크 보안 장비 (시험 단골)

👉 **실제로 막는 애들**

## 📌 IPS (침입방지시스템)

* 탐지 + 차단
* FTP / HTTP 공격 차단

---

## 📌 WAF (웹 방화벽)

* 웹 공격 방어
* SQL Injection, XSS

---

## 📌 ICMP

* 네트워크 상태 확인 (ping)

👉 💡 묶음 포인트
👉 **“트래픽 보고 / 공격 막는 계층”**

---

# 🚨 4. 공격 / 위협 유형

👉 **적이 누구냐**

## 📌 APT

* 장기 잠복 공격
* 정보 탈취 목적

👉 💡 키워드
👉 **“오래 숨어서 턴다”**

---

# 🧪 5. 보안 점검 / 진단

👉 **문제 찾는 도구**

## 📌 VAT (취약점 분석 도구)

* 시스템/네트워크 취약점 자동 탐지

👉 💡 흐름
👉 VAT → 취약점 발견 → IPS/WAF로 대응

---

## 📌 EIGRP

👉 💡 묶음
