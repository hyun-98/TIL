## 접근제어 : 누가, 무엇을, 어떤 행동을 할 수 있는지 제한하는 보안 규칙
접근제어 모델
* DAC : 임의적 접근제어 (권한을 소유자가 결정) / 파일 만든 사람이 권한 설정
* MAC : 강제 접근제어
* RBAC : 역할 기반 접근제어
* ABAC : 속성 기반 접근제어

## 인증모델 :  사용자가 누구인지 어떻게 증명하는가?

인증요소 기준
* 지식 기반 (password, PIN)
* 소유 기반 (OTP token, 스마트폰 인증앱, 보안카드)
* 생체 기반 (지문, 얼굴, 홍채)
* 행동 기반 (타이핑 패턴, 마우스 움직임)

인증 방식 모델
* 단일 인증 - password만 사용
* 다중 인증 MFA - password + 지문 / password + token

대표적인 인증 프로토콜/모델
* kerberos : 티켓 기반 인증 / server 간 인증
* OAuth : "로그인 위임", 비밀번호를 직접 넘기지 않음 / 구글 카카오
* OpenID Connect : OAuth 위에 인증기능 추가 / 로그인 + '사용자 정보 확인'
* SAML : XML 기반 인증 / 기업 SSO에서 많이 사용
* FIDO : 비밀번호 없는 인증(패스키) / 생체 + 기기 기반
