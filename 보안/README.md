## 암호학 공부 커리큘럼

암호학/보안 이론 심화 학습

* 암호학 기초 — 대칭키(AES, DES), 비대칭키(RSA, ECC), 해시(SHA), PKI
* 프로토콜 보안 — TLS/SSL, IPSec, SSH의 동작 원리
* 보안 이론 — 접근 제어, 인증 모델, 위험 분석
* 법·정책 — 개인정보보호법, ISMS-P
