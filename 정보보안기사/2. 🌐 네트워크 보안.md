
---

## 🔐 1. 핵심 개념

👉 네트워크 보안 = **네트워크를 통해 오가는 데이터 보호 + 침입 차단**

목표:

* 도청 방지
* 위변조 방지
* 비인가 접근 차단
* 서비스 정상 운영 보장

---

# 🧱 2. 네트워크 보안 기본 구조

## 📌 핵심 3요소

* 기밀성 (Confidentiality) → 정보 안 새게
* 무결성 (Integrity) → 데이터 안 바뀌게
* 가용성 (Availability) → 서비스 계속되게

👉 CIA Triad

---

# 🔥 3. 네트워크 공격 유형

## 📌 주요 공격

* DoS / DDoS → 서비스 마비
* 스니핑(Sniffing) → 패킷 도청
* 스푸핑(Spoofing) → IP/MAC 위조
* 세션 하이재킹 → 로그인 탈취
* MITM (중간자 공격) → 통신 가로채기

---

# 🧨 4. 네트워크 방어 기술

## 📌 방어 장비

* 방화벽 (Firewall) → 트래픽 차단/허용
* IDS → 침입 “탐지”
* IPS → 침입 “차단”
* WAF → 웹 공격 방어

---

## 📌 추가 방어 기술

* VPN → 암호화된 터널 통신
* NAC → 네트워크 접속 통제
* Proxy → 중계 서버

---

# 🔐 5. 암호화 기술

## 📌 대칭키

* 같은 키 사용
* 빠름
* 예: AES

## 📌 비대칭키

* 공개키 / 개인키
* 느리지만 안전
* 예: RSA

👉 핵심: 하이브리드 방식 많이 사용

---

# 🌍 6. 네트워크 계층 보안

## 📌 주요 프로토콜 보안

* SSL/TLS → 웹 암호화 (HTTPS)
* IPsec → 네트워크 계층 암호화
* SSH → 원격 접속 보안

---

# 📡 7. 네트워크 접근 통제

* 인증 (Authentication)
* 권한 (Authorization)
* 계정 기록 (Accounting)

👉 AAA 구조 (시스템 보안과 동일 개념)

---

# 🧩 8. 네트워크 보안 설계 핵심

* 최소 권한 네트워크 구성
* DMZ 구성 (외부/내부 분리)
* 세그먼트 분리 (망 분리)
* 불필요 포트 차단

---

# 📊 9. 로그 및 모니터링

* 트래픽 로그 분석
* 침입 탐지 로그
* 이상 트래픽 감지

👉 SIEM (보안 로그 통합 분석)

---

# 🔥 10. 시험 핵심 키워드 TOP 5

👉 이것만 외워도 핵심 잡힘

* DDoS
* IDS / IPS
* 방화벽
* VPN / IPsec
* 스니핑 / 스푸핑

---
