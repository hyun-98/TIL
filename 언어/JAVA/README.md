## 💡 공부 방법

❌ 문법만 계속 보기
👉 절대 비추천

✅ 이렇게 해야 됨

* 개념 10분
* 바로 코드 작성 30분

---

## ✏️ 예제 (직접 쳐보기)

```java
int sum = 0;
for (int i = 1; i <= 10; i++) {
    sum += i;
}
System.out.println(sum);
```
