# 🧱 네트워크 7계층이란?

 컴퓨터끼리 통신할 때 **7단계로 나눠서 역할을 정리한 구조**

# 🍕 쉬운 비유: 배달 앱 주문 과정

너가 배달 앱으로 피자를 주문한다고 생각해보자.


## 7️⃣ 응용 계층 (Application)

👉 우리가 직접 사용하는 단계

예:

* 웹사이트 접속
* 카카오톡 메시지

👉 “피자 주문 버튼 누르기”

---

## 6️⃣ 표현 계층 (Presentation)

👉 데이터 형식 변환

예:

* 암호화 (HTTPS)
* 이미지 변환

👉 “한국어 → 영어 번역”

---

## 5️⃣ 세션 계층 (Session)

👉 연결 유지

예:

* 로그인 상태 유지

👉 “주문 계속 이어가기”

---

## 4️⃣ 전송 계층 (Transport)

👉 데이터 잘 전달되게 관리

대표:

* TCP (정확하게)
* UDP (빠르게)

👉 “피자 빠짐없이 배달되게 확인”

---

## 3️⃣ 네트워크 계층 (Network)

👉 어디로 보내야 할지 결정

👉 IP 주소 사용

👉 “우리 집 주소 찾기”

---

## 2️⃣ 데이터 링크 계층 (Data Link)

👉 같은 네트워크 안에서 전달

👉 MAC 주소 사용

👉 “아파트 내부에서 집 찾기”

---

## 1️⃣ 물리 계층 (Physical)
👉 실제 장비, 회선과 관련된 특성 규정 (물리적 연결)
* 인코딩
* 디코딩

데이터 전송 단위 : bit
장비 : 허브, 리피터

---

# 💡 현실 예시 (웹 접속)

너가 구글 접속하면:

1. 브라우저 실행 (응용)
2. HTTPS 암호화 (표현)
3. 연결 유지 (세션)
4. 데이터 쪼개서 전송 (전송)
5. IP로 목적지 찾기 (네트워크)
6. 같은 네트워크 전달 (데이터링크)
7. 전기 신호로 전달 (물리)
